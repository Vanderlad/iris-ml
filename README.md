# iris-ml

Small ML starter project using the Iris dataset recommended to me by a T.A.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate   # mac/linux
# .venv\Scripts\activate    # windows
pip install -r requirements.txt
```
